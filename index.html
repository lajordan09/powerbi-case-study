Improving Data Visualization Accuracy in Power BI

Real Estate Market Analysis Case Study

Overview

This project analyzes residential real estate data to understand the relationship between pricing and days on market. During development, a visualization issue surfaced that required investigation, debugging, and restructuring of the data model.

The Problem

A scatter plot intended to show pricing vs. days on market displayed incorrectly. Data points were clustered and did not reflect expected market behavior.

The visualization was not reliable for decision-making due to incorrect data representation.

Root Cause

The issue was caused by Power BI using row-level data instead of aggregated data.

This resulted in multiple data points per ZIP code and distorted averages.

Solution

Created a summarized table grouped by ZIP code to ensure accurate aggregation.

  • Average List Price
  • Average Days on Market
  • Total Transactions

Rebuilt the scatter plot using the aggregated dataset to ensure one data point per ZIP code.

Outcome

The updated visualization clearly shows how pricing impacts time on market across different areas.

The chart is now accurate, readable, and useful for real estate decision-making.
